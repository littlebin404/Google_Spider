# Google_Spider
### 基于google hacking语法的多协程爬虫工具。

1. 在对应文本文件中，配置好：代理，google domain，目标domain和浏览器UA标识；

2. 设置好google hacking语法：
`site:{domain} inurl:admin','site:{domain} inurl:upload`

3. 即可爬取基于google hacking语法的网页链接，并将结果自动保存到/result/save.txt;
